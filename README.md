# DumbOS 📟

A lightweight, dumbphone-friendly dashboard and arcade OS.

**👉 [Launch DumbOS](https://skadadal.github.io/Dumbstart/)**

## Features
*   **Dash:** Quick links to lightweight tools (FrogFind, Weather, Wiki).
*   **Arcade:**
    *   🐍 **Snake:** Classic Nokia vibes.
    *   🧮 **CalcGame:** Math speed run.
    *   🧠 **PixelSimon:** Memory test.
    *   ⛳️ **DumbGolf:** 9-hole grid golf.
*   **Dumb-Ready:** Optimized for D-Pad navigation, 240px screens, and T9 keypads.
